Key Takeaways

✓Two independent frontends (PHP + Node) sharing one codebase
✓OC5 is half the size, half the steps, half the deps
✓Shared oc-frontend submodule — every JS fix benefits both
✓Template pipeline: OC4 Twig → mechanical conversion → OC5 Nunjucks
✓Deterministic deploy: one Ansible command from bare VM to running stack
✓Incremental migration possible — NPM can route pages to either stack

The database is the asset. The rest is replaceable.
